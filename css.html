CSS er et sprog som beskriver udseende af et HTML dokument. Det giver mulighed for at adskille struktur og udseende, altså at skifte udseende, ikke at skifte struktur af en HTML dokument. “Styles” er regler som kan blive navngivet og refereres i et HTML dokument. Et “StyleSheet” har alle de regler i det samme sted.

En StyleSheet kan placeres på tre forskellige måder.



CSS Syntaks

En selector er, hvad skal styles. Der er tre primære type selectors. Det kan være alle tags af en slags type (p, main, a, img, h1, osv), almindelige tags. Det kan også være class selectors. De skal bruges når der er flere elementer som skal skifte stil. Tags er mærket med en klasse (<p class=”general”>) i HTML dokument, og er brugt med punktum i stylesheet ( .general {......). Til sidst, er id selector. Det er brugt, når det er et helt specifikt element som skal skifte stil. Tags er mærket i HTML (<p id=”paragraph1”>) og i stylesheet er # hashtag brugt ( #paragraph1 {....).

Den declaration er imellem { } (curly brackets). De property eller egenskaber er hvilken karakteristik som skal skiftes, og value er, hvad skal det skiftes til. Der skal altid være en colon imellem property og value, ellers vil det ikke blive anerkendt. Desuden kan der blive mere end en enkelt property under en selector, Der kan ligger flere properties under en selector, bare husk at bruge en semi-colon mellem properties, og alle properties skal være under det samme par brackets.


Psuedo klasse selectors er et nøgleord, som er brugt til at skifte en særlig tilstand af en selector (for eksempel, hover på en button ( :hover ), eller et link som har være visited ( :visited ) før).



Cascading Regler

I CSS er kaskade meget vigtig. Det angiver hvilke regler er vigtigst, Men betydning afhænger af hvad er vigtigt, hvor specifikt det er, og kildeorden.

Hvis der er brugt !important i StyleSheet, vil det være den selector som har mest betydning til selector. De vil altid vinde over andre CSS.

Specificity betyder at dem med lav specificity, vil ikke være så vigtig i en StyleSheet. Element selectors (p, h1, main) har lav specificty. Class selectors har højere specificity over element selectors, og vil være mere vigtig i CSS. Men id selectors har den højeste specificty over resten. De vinder over alle andre.

Til sidst er Source order (kildeorder), det betyder, at hvis du har flere af de samme selectors, vil den sidst skrevet selector have mest betydning.
